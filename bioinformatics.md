## one-liner

ensemble id -> symbol -> biotype

    zcat Homo_sapiens.GRCh38.84.gtf.gz \
        | awk '$3=="gene"' \
        | perl -ne 'next unless /gene_id "(.+?)".+gene_name "(.+?)".+gene_biotype "(.+?)"/; print "$1\t$2\t$3\n";' \
        > Homo_sapiens.GRCh38.84.gtf.gz.ensemble2symol-biotype.tsv
